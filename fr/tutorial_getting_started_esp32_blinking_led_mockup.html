Bien démarrer avec l’ESP32 – Maquette LED clignotante

Un premier projet rapide pour valider votre chaîne d’outils : pilote → MicroPython → Thonny → LED clignotante.

Carte : ESP32 DevKitC (Wroom-32) • Langage : MicroPython • IDE : Thonny

Matériel requis

  • Carte ESP32 DevKitC (ESP32‑Wroom‑32)
  • Câble Micro‑USB (souvent non fourni avec la carte)
  • LED + résistance 390 Ω + fils / breadboard
  • Un ordinateur (instructions Windows ci‑dessous ; autres OS similaires)

La documentation ESP32 est disponible chez Espressif : documentation.espressif.com.

1) Pilote USB‑UART (CP210x)

La plupart des cartes ESP32 utilisent une puce pont USB‑UART, souvent une Silicon Labs CP2102/CP210x. Si le pilote correspondant manque, la carte peut ne pas apparaître en tant que port COM.

Vérifier sous Windows

  1. Ouvrez le Gestionnaire de périphériques avec Win + X, then M.
  2. Branchez l’ESP32 avec un câble Micro‑USB.
  3. Cherchez un nouveau port COM (par exemple dans « Ports (COM & LPT) »).

Si aucun nouveau port n’apparaît, installez le pilote VCP Silicon Labs :

Télécharger les pilotes VCP CP210x

Après installation, débranchez/rebranchez la carte et vérifiez que le port COM apparaît.

2) Thonny + MicroPython

Pour une maquette rapide, nous utiliserons MicroPython with the Thonny IDE. MicroPython est interprété (idéal pour itérer vite) ; pour des applications exigeantes, un environnement compilé (Arduino/ESP‑IDF) peut être préférable.

  1. Installez Thonny puis connectez l’ESP32.
  2. Dans Thonny : Outils → Options → Interpréteur.
  3. Sélectionnez MicroPython (ESP32) et le port COM de la carte.
  4. Maintenez le bouton BOOT puis cliquez sur Installer ou mettre à jour MicroPython.

3) Câblage de la LED

Connectez une LED à GPIO 16 via une résistance série de 390 Ω vers la masse :

  • GPIO16 → resistor → LED anode (+)
  • LED cathode (−) → GND

Astuce : Gardez l’USB connecté pendant le développement ; la carte peut être alimentée via USB pour ce test.

4) Le script de clignotement

Collez le script suivant dans Thonny :

from machine import Pin
		import time

		# LED on GPIO 16 (adjust if you use a different pin)
		led = Pin(16, Pin.OUT)

		while True:
			led.value(1)
			print("1")
			time.sleep(0.5)

			led.value(0)
			print("0")
			time.sleep(0.5)

5) Exécuter et valider

Cliquez sur le bouton Play dans Thonny pour exécuter le script depuis l’ordinateur.

Si tout est correctement configuré, la LED sur GPIO16 clignotera toutes les 0,5 s, confirmant le bon fonctionnement de la maquette.
