KiCad 9.0 – Build Your First PCB

A complete PCB design walkthrough in KiCad 9, covering project setup, schematic, footprint assignment, BOM generation, PCB layout, verification, design-rule checks, and Gerber manufacturing outputs.

← Back to tutorials Watch the video

Main sections covered in the video

  • Project setup – create a new KiCad project, configure backups and page settings.
  • Schematic design – set up net classes, place symbols (including external libraries), wire the circuit, and run ERC.
  • Footprint management – assign footprints, inspect them, and create/modify a custom footprint when needed.
  • Bill of Materials (BOM) – generate and export a BOM.
  • PCB layout – import from schematic, set board/layer rules, place parts, and route traces.
  • Planes, mechanics, and silkscreen – create a ground plane, add mounting holes, add a logo, and draw the board outline (edge cuts).
  • 3D visualization – assign 3D models and inspect the assembled board in the 3D viewer.
  • Verification – run DRC and fix remaining issues.
  • Manufacturing outputs – generate Gerbers and drill files and inspect them with the Gerber viewer.
  • Best practices – review common PCB pitfalls and use a design-rule checklist before fabrication.
